# 换行
Markdown换行的方式有很多种:

- 直接在一句话后敲两个空格
- 两句话之间加一个空行
- 如果你在编辑的时候，想让一行文字在显示的时候换行，就在中间加<br/>

# 列表

### 无序列表
* 项目 1
- 项目 2
+ 项目 3
  
### 有序列表

2. 第二项

### 控制层级
+ 无序列表 1
+ 无序列表 2
    + 无序列表 2.1
    	+ 无序列表 2.2  

1. 有序列表 1
	1.1 有序列表 1.1
2. 有序列表 2
	2.1 有序列表2.1
